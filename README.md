# Simple Restaurant Menu & Orders (CLI)

## Run
```bash
npm install   # (nothing to install, but keeps the workflow familiar)
npm start
# or:
node main.js
```

## Data files
- `data/categories.json`
- `data/items.json`
- `data/orders.json`

They are auto-created if missing.

## Undo Feature (Extra)
Inside **Orders** menu:
- After adding an item, you can type `undo` to revert the last addition.
- You can also choose option (5) to undo the last selection for the active order.
